# pottal-fi-bhantal-diwe
POTTAL FI BHANTAL DIWÉ est une application web de gestion communautaire conçue pour soutenir le développement et la solidarité au sein de notre village.  Cette plateforme permet de centraliser, suivre et gérer l'ensemble des contributions financières destinées aux projets communautaires, aux situations d'urgence, aux sacrifices collectifs...
